## Team Features

### Invite Team Members

1. Go to **Project Settings** → **Team**
2. Click **Invite Member**
3. Enter email address
4. Choose role:
   - **Viewer:** Can view diagrams
   - **Editor:** Can edit diagrams
   - **Admin:** Full project control
5. Click **Send Invitation**

### Roles & Permissions

**Viewer**

- View diagrams
- Add comments
- Export diagrams
- Cannot edit
- Cannot delete

**Editor**

- All viewer permissions
- Edit diagrams
- Add/remove resources
- Modify layouts
- Cannot delete project

**Admin**

- All editor permissions
- Manage team members
- Delete project
- Configure settings

## Real-Time Collaboration

### Live Updates

When team members make changes:

- Changes appear instantly
- See who's viewing/editing
- Cursor positions visible
- No refresh needed

### Presence Indicators

See who's active:

- Online status
- Current view
- Active selections
- Recent activity

### Conflict Resolution

If multiple people edit simultaneously:

- Last write wins (with notification)
- Can undo changes
- View change history
- Merge conflicts automatically

## Comments & Annotations

### Add Comments

1. Select a resource
2. Click **Comment** icon
3. Type your message
4. Tag team members with @
5. Click **Post**

### Comment Features

- **Threaded Discussions:** Reply to comments
- **Mentions:** Notify team members
- **Resolve:** Mark comments as resolved
- **Attachments:** Add screenshots or files

### Annotations

Add visual annotations:

- Text labels
- Arrows and highlights
- Custom shapes
- Color coding

## Version History

### View History

1. Click **History** in project menu
2. See all changes over time
3. View who made changes
4. See what changed

### Compare Versions

1. Select two versions
2. Click **Compare**
3. See side-by-side diff
4. Identify changes

### Restore Versions

1. Find version to restore
2. Click **Restore**
3. Confirm restoration
4. Previous version becomes current

## Sharing

### Share Links

Create shareable links:

1. Click **Share** → **Get Link**
2. Choose permissions:
   - **Public:** Anyone with link
   - **Team Only:** Team members
   - **Password Protected:** Requires password
3. Set expiration (optional)
4. Copy and share link

### Embed Diagrams

Embed in documentation:

## Troubleshooting

### Can't See Changes

**Problem:** Changes not appearing

**Solutions:**

- Refresh browser
- Check internet connection
- Verify you have permissions
- Clear browser cache

### Permission Issues

**Problem:** Can't edit diagram

**Solutions:**

- Check your role
- Ask admin for permissions
- Verify project settings
- Contact support

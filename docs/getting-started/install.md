# Install

Install Cloudeval to get started.

## Installation Methods

### Using pip

```bash
pip install cloudeval
```

### Using conda

```bash
conda install -c conda-forge cloudeval
```

## Verify Installation

After installation, verify that Cloudeval is installed correctly:

```bash
cloudeval --version
```

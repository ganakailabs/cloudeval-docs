# Assets

Place your documentation assets here.

## Required Assets

- `logo.png` - Logo for the documentation site (recommended size: 200x50px)
- `favicon.png` - Favicon for the site (recommended size: 32x32px or 16x16px)

## Optional Assets

- Images for documentation pages
- Custom CSS files (if needed)
- Other static assets

Note: If logo.png or favicon.png are not present, the site will work without them, but the logo/favicon won't be displayed.
